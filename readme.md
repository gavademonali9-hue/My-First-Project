# Project description:-
A simple and user-friendly recipe website built using HTML. This project showcases a collection of delicious recipes with easy navigation between pages like Home, About, and Contact. It is designed to provide clear recipe instructions, ingredients, and a smooth browsing experience for users.


# Setup instructions:-
1.Download or clone the repository:

git clone https://github.com/your-username/recipe-website.git

2.Open the project folder.
 
3.Double-click on index.html
OR open it in any web browser.

# Usage details:-
Open index.html in your browser to start the website.
Navigate through pages like Home, About, and Contact using links.
Browse recipes, view ingredients, and follow preparation steps easily.
